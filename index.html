Czy Ice Client jest bezpieczny?

Tak, Ice Client wykorzystuje najnowsze technologie zabezpieczające. Regularnie aktualizujemy zabezpieczenia przeciw-wykrywające.

Działa na serwerach z anti-cheatem?

Ice Client jest zaprojektowany tak, aby omijać większość systemów anti-cheat. Zalecamy jednak rozwagę na oficjalnych serwerach.

Jakie są wymagania systemowe?

Windows 10/11, 4GB RAM, Java 17+. Client działa również na Linux i macOS przez kompatybilne oprogramowanie.

Czy mogę zwrócić wersję PRO?

Oferujemy 14-dniową gwarancję zwrotu pieniędzy dla wersji PRO i LIFETIME.
